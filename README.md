# Weather-App
This is a Weather App built with Python using Tkinter, geopy, timezonefinder, pytz, and requests.

Users can enter a city name and the app shows:

Current weather condition (Sunny, Rainy, Cloudy, etc.)

Temperature, Pressure, Humidity, Wind speed

Local time of the city

Sunrise and Sunset times

Possible Future Features:

Integrated Map View

Desktop Weather Widget

Audio Alerts

Play warning sounds for severe conditions
(storms, dust, heatwaves, snow alerts).
